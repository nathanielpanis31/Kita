REACT AND EXPRESS SETUPS AND INSTALL

---REACT SETUP---
1.download node on computer
2.npm create vite@latest appName
3.cd appName (change directory)
4.npm install
5.npm run dev (run the web)




---REACT ADDITIONAL INSTALL---
1.npm install react-router-dom (for navigating or goin in another file on web, replacement on a href from html to avoid page refresh and use react-router-dom for navigation.)




---EXPRESS SETUP---
1.mkdir express-foldername/kita-server(making new folder inside the workspace folder)
2.cd express-foldername (change directory to new folder)
3.npm init -y (Initializes a new Node.js project in your folder. It creates a package.json file which tracks your project info, scripts, and dependencies)
4.npm install express mongoose dontenv cors(you can install this 4 separately)
   -pm install express (install express frameform)
   -pm install mongoose (the bridge between the code and the database mongoDB)
   -pm install dotenv (It loads your .env file into your app so you can access secret values like passwords and API keys without hardcoding them in your code)
   -pm install cors (Allows your React app (different port) to talk to your Express server)
5.npm install --save-dev nodemon
6. create the server.js and .env file manually
7. put the a starter code on the server.js, you can find it on code-note file its titles SERVER.JS




---MONNGODB SETUP---
1. download mongoDB, mongoDB atlas, mongoDB shell
2. download mongoDB extension on visual studio, to access mongoDB shell inside VS




---NOTES--- 
1. You have to run npm run dev on both express and react directory use a 2 shell to run them.
2. if gonna clone or pull the project on another device, cd on each directory of back and front end then install(npm install) on both, then add .env file on the express directory.
3. when starting new project always put gitignore on the root directory.
4. always make sure that your system have security like pass is ecrypt when saved on database(npm install bcrypt)
